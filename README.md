# UTD-Deep-Dive-AI-Project

Project Name: Olive Branch

Project Description:
The goal of our project is to provide the best possible environmental solution for a place which is lacking adequate greenery. The model will do this by measuring the improvements in a predetermined set of metrics (such as greenery, air quality, temperature, budget etc.) before and after the solution is implemented. Then it will settle on a solution that optimizes the metrics to the greatest possible extent.
